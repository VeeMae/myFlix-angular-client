getFaveMovies

  • getFaveMovies(): void
    • Defined in app/user-profile/user-profile.component.ts:62

    Function to retrieve user's favorite movies. The function first fetches all the movies from the database and adds them to an array called movies. Another function is called inside, which retrieves the list of movie Ids that the user has already added to their list of favorites and add it to an array called faveMovieIds. A function is then called to filter all the movies by movie Ids that match the user's list of movie Ids. A new array called faveMovies then stores the filtered movies to display only the user's list of favorites.

    Returns void
